# Lord Of Heroes Voice Archive

반갑습니다, 로드 여러분!

초호화 성우진이 도망갈까 아쉬워 기나긴 프로젝트를 시작하게 된 고사리/개사료 (@gaesaryeo) 입니다.

본 프로젝트는 순전히 개인의 조바심과, 무모함이 낳은 1인 프로젝트이므로, 게임이 셧다운되는 그날까지 얼마나 남았는지도 알 수 없지만 최대한 개인이 가지고 있는 영웅 모두를 아카이빙하는 것을 최우선으로 합니다.
제가 여유가 있어 제게 없는 영웅을 추가하기로 하고 요청하기 전까지 개별적인 아카이빙 요청은 죄송하지만 받지 못하며,
남은 시간 동안 그저 최선을 다하겠다는 말씀만을 드릴 수 있을 것 같습니다.

여러 사람이 나눠서 작업을 해 주더라도, 결국 서버에 옮기고, 양식에 맞게 코딩을 하는 것은 1인 작업이기에 아직은 요청과 도움 모두 생각지 않고 있는 점 널리 양해해 주시면 감사하겠습니다.

머무시는 동안 눈물 찔끔 한 방울, 여운 남는 웃음 가지고 돌아가시를 바라며
우리의 걱정이 기우가 되도록 긴 날 아주 오랜 날 이 게임을 플레이할 수 있었으면 좋겠습니다.

# LOH Voice Room Archive 도움 설명서

안녕하세요, 개사료 @gaesaryeo 입니다.

빠른 자료 확보로 미연에 일어날 수 있는 어떤 일이라도 방지하고자 도움을 요청하게 되었습니다.

저는 음성과 음량을 일정하게 유지하기 위해 다음과 같은 과정을 거쳐 아카이브하고 있습니다. 가급적 이 방법으로 시도해 주신다면 더 많은 데이터가 들쭉날쭉한 컨디션 없이 고음질로 보존될 수 있으니 번거로우시더라도 도움의 손길을 뻗어주실 의향이 있다면 이 절차대로 아카이빙을 진행해 주시면 감사하겠습니다.

## 필요 설치 프로그램: [Moo0 음성 녹음기](https://kor.moo0.com/software/VoiceRecorder/), 블루스택 또는 모바일 게임 플레이어

1. 먼저 블루스택에서 로드오브히어로즈를 실행해 줍니다.  
2. 볼륨을 적당히 조절한 후, Moo0 음성 녹음기를 켜 줍니다.  
3. 조그만 인터페이스에서, 설정을 누릅니다.

![][image1]  
설정 \> Mp3 녹음 음질 \> 최상 음질 (320kbps)  
설정 \> 파일 명명, 개명 방식 \> ‘(1), (2), (3) 추가 (중복 시)’ 체크  
‘무음 공간 삭제’ ‘모든 PC 사운드’ ‘부스트’ 40%

4. 기본적인 설정은 끝났습니다. 무음 삭제 기능이 자동으로 있기 때문에 필요한 대사만 딸 수 있습니다. 무음구간을 별도로 편집할 필요 없이, 자동으로 번호가 (1), (2), (3) 으로 매겨지기 때문에 편합니다.  
5. 하나의 경로에 5속성 영웅 혹은 본인이 아카이빙하신 영웅을 정리해주세요.  
   \-예시: joshua/light\_joshua.mp3 (속성은 꼭 적어주세요)  
6. 압축파일을 [hyeya4847@gmail.com](mailto:hyeya4847@gmail.com) 으로 보내주시면 됩니다. \[제목 양식: 속성/영웅이름\]
